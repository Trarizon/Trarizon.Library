# Trarizon.Library.CodeAnalysis

This library contains types used for analyzing code, to generate template codes or improve robustness

## Contents

Type name with in brackets are attribute types

### Diagnostics

Type|Summary
:-:|:--
`[ExternalSealed]`| Only allow inherit type or implement interface in current assembly
`[FriendAccess]`| Only specified types can access the member

### Generation

Type|Summary
:-:|:--
`[Singleton]`| Generate singleton
